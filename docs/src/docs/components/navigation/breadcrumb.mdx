# Breadcrumb

Display the current page's location in the navigation hierarchy.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "breadcrumb",
  "items": [
    { "label": "Home", "href": "/" },
    { "label": "Docs", "href": "/docs" },
    { "label": "Components", "current": true }
  ]
}
```
</CodeBlock>

## Properties

- `items` - Array of breadcrumb items
- `separator` - Custom separator (default: chevron)

## Examples

### With Icons

<CodeBlock lang="json">
```json
{
  "type": "breadcrumb",
  "items": [
    { "label": "Home", "href": "/", "icon": "home" },
    { "label": "Current Page", "current": true }
  ]
}
```
</CodeBlock>
