# Tabs

Organize content into tabbed sections.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "tabs",
  "items": [
    { "label": "Tab 1", "value": "tab1" },
    { "label": "Tab 2", "value": "tab2" }
  ],
  "children": [
    {
      "type": "tabPanel",
      "value": "tab1",
      "children": [{ "type": "text", "content": "Content 1" }]
    },
    {
      "type": "tabPanel",
      "value": "tab2",
      "children": [{ "type": "text", "content": "Content 2" }]
    }
  ]
}
```
</CodeBlock>

## Properties

- `items` - Array of tab definitions
- `defaultValue` - Initially selected tab
- `children` - Tab panel components
