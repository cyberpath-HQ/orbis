# Tooltip

Display contextual information on hover.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "tooltip",
  "content": "This is helpful information",
  "children": [
    { "type": "button", "text": "Hover me" }
  ]
}
```
</CodeBlock>

## Properties

- `content` - Tooltip text
- `position` - Tooltip position (top, bottom, left, right)
- `children` - Trigger element

## Examples

### With Icon

<CodeBlock lang="json">
```json
{
  "type": "tooltip",
  "content": "Click for more details",
  "position": "right",
  "children": [
    { "type": "icon", "name": "info" }
  ]
}
```
</CodeBlock>
