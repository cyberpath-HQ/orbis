# Dialog

Display content in a modal overlay dialog.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "dialog",
  "title": "Confirm Action",
  "open": "${state.dialogOpen}",
  "children": [
    { "type": "text", "content": "Are you sure?" }
  ],
  "actions": [
    { "type": "button", "text": "Cancel", "on_click": [] },
    { "type": "button", "text": "Confirm", "variant": "primary", "on_click": [] }
  ]
}
```
</CodeBlock>

## Properties

- `title` - Dialog title
- `open` - Dialog visibility state
- `children` - Dialog content
- `actions` - Footer action buttons
- `size` - Dialog size (sm, md, lg)
