# Progress

Display loading progress and completion status.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "progress",
  "value": 60,
  "max": 100
}
```
</CodeBlock>

## Properties

- `value` - Current progress value
- `max` - Maximum value (default: 100)
- `label` - Optional label text
- `variant` - Style variant

## Examples

### With Label

<CodeBlock lang="json">
```json
{
  "type": "progress",
  "value": "${state.uploadProgress}",
  "max": 100,
  "label": "Uploading... ${state.uploadProgress}%"
}
```
</CodeBlock>
