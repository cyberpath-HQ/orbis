# Skeleton

Display loading placeholders while content is being fetched.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "skeleton",
  "width": "100%",
  "height": "20px"
}
```
</CodeBlock>

## Properties

- `width` - Skeleton width
- `height` - Skeleton height
- `variant` - Shape variant (text, circular, rectangular)

## Examples

### Card Skeleton

<CodeBlock lang="json">
```json
{
  "type": "stack",
  "children": [
    { "type": "skeleton", "variant": "circular", "width": "40px", "height": "40px" },
    { "type": "skeleton", "width": "100%", "height": "20px" },
    { "type": "skeleton", "width": "80%", "height": "20px" }
  ]
}
```
</CodeBlock>
