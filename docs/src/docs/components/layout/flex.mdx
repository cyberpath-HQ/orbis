# Flex

The Flex component provides a flexible box layout with full control over alignment and distribution.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "flex",
  "direction": "row",
  "justify": "space-between",
  "align": "center",
  "children": [
    // Flex items
  ]
}
```
</CodeBlock>

## Properties

- `direction` - Flex direction: "row", "column", "row-reverse", "column-reverse"
- `justify` - Justify content: "start", "center", "end", "space-between", "space-around"
- `align` - Align items: "start", "center", "end", "stretch"
- `wrap` - Enable wrapping: true/false
- `gap` - Spacing between items
