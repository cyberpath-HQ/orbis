# Stack

Stack components arrange children vertically or horizontally with consistent spacing.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "stack",
  "direction": "vertical",
  "spacing": "md",
  "children": [
    // Stack items
  ]
}
```
</CodeBlock>

## Properties

- `direction` - Layout direction: "vertical" or "horizontal"
- `spacing` - Space between items (xs, sm, md, lg, xl)
- `align` - Alignment (start, center, end, stretch)

## Examples

### Vertical Stack

<CodeBlock lang="json">
```json
{
  "type": "stack",
  "direction": "vertical",
  "spacing": "sm",
  "children": [
    { "type": "text", "content": "First item" },
    { "type": "text", "content": "Second item" }
  ]
}
```
</CodeBlock>
