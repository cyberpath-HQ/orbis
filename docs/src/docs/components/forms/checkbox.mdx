# Checkbox

Checkbox for boolean selections and multi-select options.

## Basic Usage

<CodeBlock lang="json">
```json
{
  "type": "checkbox",
  "name": "terms",
  "label": "I accept the terms and conditions",
  "checked": "${state.acceptedTerms}"
}
```
</CodeBlock>

## Properties

- `name` - Checkbox field name
- `label` - Label text
- `checked` - Checked state
- `disabled` - Disable the checkbox
- `on_change` - Action on state change

## Examples

### Checkbox Group

<CodeBlock lang="json">
```json
{
  "type": "stack",
  "children": [
    { "type": "checkbox", "name": "feature1", "label": "Feature 1" },
    { "type": "checkbox", "name": "feature2", "label": "Feature 2" },
    { "type": "checkbox", "name": "feature3", "label": "Feature 3" }
  ]
}
```
</CodeBlock>
