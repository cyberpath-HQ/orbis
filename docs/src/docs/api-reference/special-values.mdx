## Special Values

Orbis provides several built-in variables and special values available in expressions.

## Event Variables

### $value

The current value in change events.

<CodeBlock lang="json">
```json
{
  "type": "Field",
  "events": {
    "on_change": [
      { "type": "update_state", "path": "inputValue", "value": "{{$value}}" }
    ]
  }
}
```
</CodeBlock>

**Available in:** `on_change`, `on_select`, `onSort`, `on_page_change`, `onTabChange`

### $event

The full event object.

<CodeBlock lang="json">
```json
{
  "events": {
    "on_click": [
      { "type": "update_state", "path": "clickX", "value": "{{$event.clientX}}" }
    ],
    "on_key_down": [
      {
        "type": "conditional",
        "condition": "{{$event.key === 'Enter' && $event.ctrlKey}}",
        "then": [{ "type": "call_api", "api": "submit" }]
      }
    ]
  }
}
```
</CodeBlock>

**Properties:**

| Property | Type | Description |
|----------|------|-------------|
| `$event.key` | string | Key pressed (keyboard events) |
| `$event.ctrlKey` | boolean | Ctrl modifier |
| `$event.shiftKey` | boolean | Shift modifier |
| `$event.altKey` | boolean | Alt modifier |
| `$event.metaKey` | boolean | Meta/Cmd modifier |
| `$event.clientX` | number | Mouse X (mouse events) |
| `$event.clientY` | number | Mouse Y (mouse events) |
| `$event.target` | object | Event target element |

## Loop Variables

### $item

Current item in a Loop component.

<CodeBlock lang="json">
```json
{
  "type": "Loop",
  "items": "{{state.users}}",
  "render": {
    "type": "Text",
    "text": "Name: {{$item.name}}, Email: {{$item.email}}"
  }
}
```
</CodeBlock>

### $index

Current index in a Loop component.

<CodeBlock lang="json">
```json
{
  "type": "Loop",
  "items": "{{state.items}}",
  "render": {
    "type": "Text",
    "text": "{{$index + 1}}. {{$item.title}}"
  }
}
```
</CodeBlock>

**Note:** Index is 0-based.

### Custom Loop Variables

Use `itemAs` and `indexAs` for custom names:

<CodeBlock lang="json">
```json
{
  "type": "Loop",
  "items": "{{state.products}}",
  "itemAs": "product",
  "indexAs": "i",
  "render": {
    "type": "Text",
    "text": "{{i + 1}}. {{product.name}}"
  }
}
```
</CodeBlock>

## API Response Variables

### $response

Full response from API calls.

<CodeBlock lang="json">
```json
{
  "type": "call_api",
  "api": "getUsers",
  "on_success": [
    { "type": "update_state", "path": "users", "value": "{{$response.data}}" },
    { "type": "update_state", "path": "total", "value": "{{$response.total}}" }
  ]
}
```
</CodeBlock>

**Common properties:**

| Property | Type | Description |
|----------|------|-------------|
| `$response` | any | Full response |
| `$response.data` | any | Response data |
| `$response.items` | array | List items |
| `$response.total` | number | Total count |
| `$response.page` | number | Current page |
| `$response.pages` | number | Total pages |

### $error

Error object from failed operations.

<CodeBlock lang="json">
```json
{
  "type": "call_api",
  "api": "saveData",
  "on_error": [
    { "type": "show_toast", "message": "Error: {{$error.message}}", "level": "error" },
    { "type": "update_state", "path": "error", "value": "{{$error}}" }
  ]
}
```
</CodeBlock>

**Properties:**

| Property | Type | Description |
|----------|------|-------------|
| `$error.message` | string | Error message |
| `$error.code` | string | Error code |
| `$error.status` | number | HTTP status |
| `$error.details` | object | Additional details |

## Navigation Variables

### params

Route parameters from dynamic routes.

<CodeBlock lang="json">
```json
// Route: /users/:id
{
  "on_mount": [
    { "type": "call_api", "api": "getUser", "params": { "id": "{{params.id}}" } }
  ]
}
```
</CodeBlock>

### query

Query string parameters.

<CodeBlock lang="json">
```json
// URL: /search?q=test&page=2
{
  "on_mount": [
    { "type": "update_state", "path": "searchQuery", "value": "{{query.q}}" },
    { "type": "update_state", "path": "page", "value": "{{query.page || 1}}" }
  ]
}
```
</CodeBlock>

### route

Current route information.

<CodeBlock lang="json">
```json
"{{route.path}}"    // Current path
"{{route.hash}}"    // URL hash
```
</CodeBlock>

## Time Variables

### $now

Current timestamp.

<CodeBlock lang="json">
```json
{
  "type": "Text",
  "text": "Current time: {{$now}}"
}
```
</CodeBlock>

<CodeBlock lang="json">
```json
{
  "type": "download",
  "url": "/api/export",
  "filename": "export-{{$now}}.csv"
}
```
</CodeBlock>

## Window Variables

### $window

Browser window object.

<CodeBlock lang="json">
```json
{
  "type": "Conditional",
  "conditions": [
    {
      "when": "{{$window.innerWidth < 768}}",
      "render": { "type": "Text", "text": "Mobile view" }
    }
  ]
}
```
</CodeBlock>

**Properties:**

| Property | Type | Description |
|----------|------|-------------|
| `$window.innerWidth` | number | Viewport width |
| `$window.innerHeight` | number | Viewport height |
| `$window.location.origin` | string | Origin URL |
| `$window.location.pathname` | string | Current path |
| `$window.location.href` | string | Full URL |

## Form Variables

### form

Form data namespace.

<CodeBlock lang="json">
```json
"{{form.myFormId}}"              // Complete form data
"{{form.myFormId.fieldName}}"    // Specific field
"{{form.myFormId.$valid}}"       // Validation state
"{{form.myFormId.$dirty}}"       // Modified state
"{{form.myFormId.$errors}}"      // All errors
"{{form.myFormId.$errors.email}}" // Field error
```
</CodeBlock>

## Loading Variables

### loading

Loading states namespace.

<CodeBlock lang="json">
```json
"{{loading.submit}}"     // Submit loading
"{{loading.fetch}}"      // Fetch loading
"{{loading.keyName}}"    // Any loading key
```
</CodeBlock>

## State Variables

### state

Plugin state namespace.

<CodeBlock lang="json">
```json
"{{state.fieldName}}"        // Root field
"{{state.user.name}}"        // Nested field
"{{state.items[0]}}"         // Array index
"{{state.items.length}}"     // Array length
```
</CodeBlock>

## Plugin Variables

### plugin

Plugin-specific variables.

<CodeBlock lang="json">
```json
"{{plugin.id}}"           // Plugin ID
"{{plugin.version}}"      // Plugin version
"{{plugin.name}}"         // Plugin name
```
</CodeBlock>

## Environment Variables

### env

Environment configuration.

<CodeBlock lang="json">
```json
"{{env.API_URL}}"         // API URL
"{{env.DEBUG}}"           // Debug mode
```
</CodeBlock>

## Usage Examples

### Complete Form Submit

<CodeBlock lang="json">
```json
{
  "type": "Button",
  "text": "{{loading.submit ? 'Saving...' : 'Save'}}",
  "loading": "{{loading.submit}}",
  "disabled": "{{!form.userForm.$valid || loading.submit}}",
  "events": {
    "on_click": [
      { "type": "set_loading", "key": "submit", "value": true },
      {
        "type": "call_api",
        "api": "saveUser",
        "params": { "data": "{{form.userForm}}" },
        "on_success": [
          { "type": "show_toast", "message": "Saved {{$response.name}}!", "level": "success" }
        ],
        "on_error": [
          { "type": "show_toast", "message": "{{$error.message}}", "level": "error" }
        ],
        "onComplete": [
          { "type": "set_loading", "key": "submit", "value": false }
        ]
      }
    ]
  }
}
```
</CodeBlock>

### Dynamic List

<CodeBlock lang="json">
```json
{
  "type": "Loop",
  "items": "{{state.users.filter(u => u.active)}}",
  "itemAs": "user",
  "indexAs": "idx",
  "render": {
    "type": "Card",
    "children": [
      { "type": "Text", "text": "{{idx + 1}}. {{user.name}}" },
      { "type": "Text", "text": "Email: {{user.email}}" },
      {
        "type": "Button",
        "text": "View",
        "events": {
          "on_click": [
            { "type": "navigate", "route": "/users/{{user.id}}" }
          ]
        }
      }
    ]
  }
}
```
</CodeBlock>

### Responsive Layout

<CodeBlock lang="json">
```json
{
  "type": "Conditional",
  "conditions": [
    {
      "when": "{{$window.innerWidth >= 1024}}",
      "render": { "type": "Flex", "direction": "row", "children": [] }
    },
    {
      "when": "{{$window.innerWidth >= 768}}",
      "render": { "type": "Flex", "direction": "row", "wrap": true, "children": [] }
    }
  ],
  "fallback": { "type": "Flex", "direction": "column", "children": [] }
}
```
</CodeBlock>
